##### 拉最新代码流程
##### (1)、进入目标路径文件夹如pycharmworkspace，鼠标右键点击git bash ，进入命令窗
##### (2)、将工程代码下载到本地
##### git clone 工程地址
##### (3)、进入本地文件夹
##### cd airtestUI/
##### (4)、切换到分支
######  ![](/images/blog/gitthree.png)
###### git checkout dev,则拉取的为最新分支代码
##### 提交代码流程：（在同样文件下操作如上）
###### 1、# 先同步分支最新代码，如果提示有错误，或者冲突，需要解决冲突后再次提交  git pull
###### 2、# git add testcase/..(具体修改文件)
###### 3、git commit -m  “提交时的注释信息”
###### 4、git push  推送到远程分支
